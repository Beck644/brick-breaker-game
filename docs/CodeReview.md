1. 头文件与实现文件分离，降低编译耦合度，提升编译效率；

### 2.1 碰撞检测逻辑
1. 球与砖块碰撞：使用Raylib原生CheckCollisionCircleRec()函数，通过球中心+半径与砖块矩形的交集检测，逻辑正确。
2. 球与挡板碰撞：复用上述碰撞函数，挡板矩形与球的碰撞判断逻辑一致，结果正确。
3. 球与窗口边界碰撞：通过坐标判断+速度反向实现反弹，上下左右边界均无穿墙问题，逻辑无漏洞。
4. 砖块击破逻辑：通过isActive标记控制砖块渲染状态，击破后分数累加（每块+5），逻辑正确。
### 2.2 游戏核心功能
1. 挡板控制：支持鼠标拖拽和键盘左右键控制，响应及时无卡顿，功能完成。
2. 分数系统：击破砖块实时加分，界面显示当前分数，游戏结束后保留最终分数，功能完成。
3. 生命值系统：初始生命值3，球掉落时扣1血，生命值≤0触发游戏结束，功能完成。
4. 游戏流程：完整实现「开始→运行→结束→重启」的闭环，状态切换无异常，功能完成。
